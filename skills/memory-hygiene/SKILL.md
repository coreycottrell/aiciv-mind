# Memory Hygiene Protocol

## After Completing Significant Work
1. Write a memory_type="learning" for any pattern discovered
2. Write a memory_type="decision" for any architectural choice made
3. Pin critical identity memories with pin_memory()
4. Use memory_type="handoff" to preserve session context for the next session

## Memory Types
- learning: patterns, discoveries, how-to knowledge
- decision: architectural choices with rationale
- error: mistakes and how they were fixed (prevents repetition)
- handoff: session summary for continuity
- observation: noteworthy things noticed without action taken
- identity: foundational facts about who this mind is

## Frequency
- Write memories DURING work, not just at the end
- A memory with 3+ accesses develops high depth score — write searchable titles
- Search before writing: avoid duplicates (memory_search first)
